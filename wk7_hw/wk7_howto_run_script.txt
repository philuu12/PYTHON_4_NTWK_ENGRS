1) How to run Python Ansible module playbook, at prompt:
ansible-playbook wk7_ex3_add_vlan.yml
(wk7_ex3_add_vlan.yml is the yaml file)

2) Must have the yaml file and Ansible Python module. Ansible Python module
must reside under sub directory "library" relative to the current directory.

Ansible Python module: /home/pluu/ANSIBLE/library/ex3_add_vlan_arista.py
Yaml file in current directory: /home/pluu/ANSIBLE/wk7_ex3_add_vlan.yml
